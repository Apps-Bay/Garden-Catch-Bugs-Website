Garden Catch Bugs

Try to catch bugs in garden! In this game you need to catch bugs for gaining points to win.

Be careful, do not touch bad bugs that causing you losing points! bugs for gaining points (Bee +3, Lady bug +2, Leaf beetle +1) bugs for losing points (Stink beetle -3, Star beetle -2, Blue beetle -1)

快来花园里抓昆虫吧！在游戏里你需要抓到足够多的昆虫赢得积分。

但小心，抓到有些昆虫是会让你损失分数的，最好不要碰到他们！得分昆虫（蜜蜂 +3，瓢虫 +2， 叶甲虫 +1）失分昆虫（臭虫 -3， 星甲虫 -2，蓝甲虫 -1）
